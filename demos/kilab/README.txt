KILAB demosida

Det här är en färdig statisk hemsida för Katrineholms Industrilackering AB. Sidan är skapad som en ActyWeb-demo och är inte företagets officiella hemsida.

Filerna

- index.html innehåller sidans text och struktur.
- style.css innehåller färger, layout och mobilanpassning.
- script.js innehåller mobilmenyn.
- assets/images/ innehåller bilder och logotyp från den befintliga sparade sidan.
- screenshots/ innehåller fem bilder som kan användas när sidan visas för kunden.

Öppna sidan lokalt

Dubbelklicka på index.html. Sidan öppnas då i webbläsaren.

Vad är en domän?

En domän är webbadressen till en hemsida, till exempel kilab.se. Domänavgift betalas normalt direkt till den leverantör där domänen köps.

Vad är hosting?

Hosting är platsen där hemsidans filer ligger så att andra kan öppna sidan på internet. Den här sidan är byggd med vanlig HTML, CSS och JavaScript och kan ligga på GitHub Pages eller ett vanligt webbhotell.

Publicera på GitHub Pages

1. Lägg filerna i ett GitHub-repository.
2. Gå till repositoryts Settings.
3. Öppna Pages.
4. Välj branch main och rätt mapp.
5. Spara. GitHub visar en länk när sidan är publicerad.

Publicera på vanligt webbhotell

Ladda upp index.html, style.css, script.js och assets-mappen till webbhotellets public_html-, www- eller motsvarande mapp.

Kunden får använda filerna själv

Filerna kan lämnas över om kunden vill hantera publicering och ändringar själv. Kunden är inte låst till ActyWeb.

ActyWeb kan hjälpa till

ActyWeb kan hjälpa till med mindre ändringar, anpassning av text och bilder, publicering och domänkoppling mot en mindre hjälpavgift. Eventuell domänavgift betalas av kunden direkt till domänleverantören.

Kort sammanfattning att skicka till kunden

Jag har gjort en färdig hemsida för KILAB som samlar industrilackering, blästring, design och dekor, bilder och kontaktvägar på ett modernare och mer mobilvänligt sätt. Den är skapad som en demo och kostar inget att titta på.

Styrkor med sidan

- Tydlig första vy med telefonnummer direkt.
- Riktig text från befintlig information.
- Tydligare tjänsteöversikt än den gamla sidan.
- Kontaktpersoner, telefonnummer, e-post och adress samlade på ett ställe.
- Mobilanpassad layout.

Det kunden kan komplettera med

- Öppettider.
- Fler aktuella bilder från produktionen.
- Eventuella uppdaterade tjänstebeskrivningar.
- Information om certifikat eller kvalitetsarbete om kunden vill visa det.
- Bekräftelse på att alla kontaktuppgifter fortfarande stämmer.
